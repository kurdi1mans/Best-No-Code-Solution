# Grist
**Category:** Open-source spreadsheet/database

- **Summary:** Open-source spreadsheet with relational tables, computed columns, and Python formulas; runs self-hosted or on Grist Cloud.
- **Strengths:**
  - Relational tables with references and lookups, more structured than standard spreadsheets.
  - Python-formula support enables powerful transformations beyond typical sheet functions.
  - Self-hosting for data control; ACLs per table/column, and granular views.
- **Limitations:**
  - UI less polished than Airtable; fewer turnkey templates and integrations.
  - Automation ecosystem is lighter; often relies on API/webhooks for workflows.
  - No built-in app/front-end builder; external tools needed for portals.
- **Best for:** Teams wanting an open-source, programmable spreadsheet/DB hybrid with self-hosting and Python formulas.
- **Pricing snapshot:** Free OSS/self-host; Grist Cloud with paid tiers for users, storage, and collaboration features.
- **Notable integrations:** REST API, webhooks, Python formulas, Zapier/Make connectors (via API), embeddings.

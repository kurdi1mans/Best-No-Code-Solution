# Notion
**Category:** Docs/knowledge base with lightweight database features

- **Summary:** Collaborative docs with linked databases used for knowledge bases, lightweight CRMs, and internal portals.
- **Strengths:**
  - Flexible pages + databases enable many structures without formal schema design.
  - Good collaboration, comments, and permissions for teams.
  - Public pages for simple sites and hubs; growing API ecosystem.
- **Limitations:**
  - Performance can lag on large databases; limited automation natively.
  - External-facing experiences (portals, sites) need third-party wrappers.
  - API rate limits and eventual consistency can complicate integrations.
- **Best for:** Internal documentation, lightweight project/CRM tracking, and starter knowledge bases.
- **Pricing snapshot:** Free for individuals; team tiers for guests, permissions, and higher limits.
- **Notable integrations:** Notion API, Zapier/Make, synced databases, third-party site builders (Super, Potion), AI features.

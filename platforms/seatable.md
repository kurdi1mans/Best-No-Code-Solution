# SeaTable
**Category:** Database/spreadsheet hybrid (self-host/cloud)

- **Summary:** Airtable-style database with self-hosted and cloud options, supporting large datasets and automation scripts.
- **Strengths:**
  - Self-hosting for data control; scalable storage limits vs. Airtable caps.
  - Views, forms, automations, and Python scripts for advanced processing.
  - Plugin system for charts, maps, Kanban, galleries, and more.
- **Limitations:**
  - UI less polished than Airtable; some features lag in usability.
  - Smaller integration ecosystem; may rely on API/webhooks for many connectors.
  - Self-hosting adds ops overhead unless using managed cloud.
- **Best for:** Teams needing Airtable-like UX with higher data limits and self-hosting/governance options.
- **Pricing snapshot:** Community self-hosted free; paid cloud/on-prem tiers add users, automations, and support.
- **Notable integrations:** REST API, webhooks, Python scripts, Zapier/Make connectors, plugins marketplace.
